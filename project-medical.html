Agentic Multimodal Medical Assistant

Role: Machine Learning Engineer · Graduate Research Assistant
Timeline: Oct 2025 – Present

Project Overview

The Agentic Multimodal Medical Assistant is a privacy-preserving clinical intelligence platform designed to assist healthcare professionals in real-time decision making. The system integrates clinical text data (EHR notes, reports) with medical imaging data using multimodal AI techniques while ensuring strict patient data privacy through federated learning.

This project addresses key challenges in modern healthcare AI, including data silos, privacy regulations, and the need for context-aware medical decision support systems.

Problem Statement

Traditional medical AI systems rely on centralized datasets, which creates serious privacy risks and limits data sharing across institutions. Additionally, unimodal models fail to capture the full clinical context when text and images are treated independently.

The challenge was to design a system that:

  • Preserves patient privacy
  • Leverages both clinical text and medical images
  • Supports real-time clinical decision making
  • Reduces false negatives in high-risk scenarios

System Architecture & Workflow

  • Clinical text and medical images are processed locally at each hospital node
  • Text data is embedded using transformer-based language models
  • Medical images are encoded using vision models
  • Multimodal feature fusion aligns image-text representations
  • Federated learning aggregates model updates without sharing raw data
  • Central coordinator updates global model parameters

This architecture ensures that sensitive patient data never leaves its source environment while still enabling collaborative model improvement.

Modeling Approach

The system employs multimodal learning techniques to jointly represent visual and textual medical information. Feature fusion strategies were applied to combine embeddings from vision and language models, enabling contextual reasoning across modalities.

Federated learning was implemented to support decentralized training across multiple medical institutions while complying with privacy regulations such as HIPAA.

Evaluation Strategy

Given the high-risk nature of medical applications, evaluation focused on minimizing false negatives rather than maximizing raw accuracy.

  • Precision-Recall AUC (PR-AUC)
  • Recall-focused metrics
  • Error analysis on critical clinical cases

This ensured that the system prioritizes patient safety and clinical reliability.

Impact & Outcomes

The system demonstrates the feasibility of deploying privacy-preserving multimodal AI in healthcare environments. It improves diagnostic support, enhances clinical context understanding, and reduces risks associated with centralized data storage.

The approach can be extended to other sensitive domains such as finance and government analytics.

Technology Stack

Python · PyTorch · Transformers · NLP · Medical Imaging · Multimodal AI · Federated Learning · Data Privacy Techniques
