Digital Archive AI Platform

Role: Research Data Scientist · Graduate Assistant

Project Overview

This project focuses on improving metadata quality for large-scale digital archives using OCR, embeddings, and LLM-driven optimization techniques. The platform enhances accuracy, completeness, and semantic consistency of archival metadata.

Problem Statement

Digital archives often contain incomplete, inconsistent, or poorly structured metadata, which severely impacts search, retrieval, and usability. Manual metadata correction is time-consuming and error-prone.

System Workflow

  • OCR extracts text from scanned documents
  • Text embeddings detect metadata gaps
  • LLM prompts enrich and normalize metadata
  • Vector similarity improves retrieval

Evaluation & Impact

The system improved metadata consistency and significantly enhanced search relevance across large digital collections.

Tech Stack

Python · OCR · GPT-based LLMs · Embeddings · Vector Databases
